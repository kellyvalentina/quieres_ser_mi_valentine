# ¿Quieres ser mi Valentine?

A cute interactive Valentine's Day website built with HTML, CSS and vanilla JavaScript. It walks you through several sections with love notes, lets you pick your answer, and ends with a virtual letter you can customize and download as an image — saved in soft pink tones.

**[Live site →](https://kellyvalentina.github.io/quieres_ser_mi_valentine/)**

---

## Features

- Multi-section layout with smooth navigation between pages
- Interactive button flow that guides the user through the experience
- Virtual letter builder with downloadable output
- Downloaded card rendered in a soft pink color palette
- Fully client-side — no backend, no dependencies
